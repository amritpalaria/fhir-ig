# Terminology

## Code Systems

* Name: [Attachment Tag CodeSystem](CodeSystem-attachment-tag-cs.md)
  * Description: Codes used for semantic tagging and categorization of attachments.
* Name: [Observation Tag CodeSystem](CodeSystem-observation-tag.md)
  * Description: Codes used to qualify DiagnosticReport and Observation resources within hierarchical parent-child result/report structures.
* Name: [314e Procedure Category CodeSystem](CodeSystem-procedure-category-314e.md)
  * Description: Hierarchical procedure categories used for operational, workflow, and semantic classification of Procedure resources.
* Name: [314e Time Precision Units](CodeSystem-time-precision-units-cs.md)
  * Description: Time precision units used to indicate the actual precision of a dateTime, time, or instant value.

## Value Sets

* Name: [Attachment Tag ValueSet](ValueSet-attachment-tag.md)
  * Description: Standard semantic tags used for categorization and workflowqualification of attachments.
* Name: [314e Lab Subcategories](ValueSet-lab-category-subcategory-314e.md)
  * Description: Procedure subcategories of Lab category used for more granularoperational classification.
* Name: [Observation Tag ValueSet](ValueSet-observation-tag-vs.md)
  * Description: Semantic tags used in meta.tag to qualify DiagnosticReport andObservation resources participating in hierarchical parent-childresource structures.
* Name: [314e Broad Procedure Categories](ValueSet-procedure-category-broad-314e.md)
  * Description: Top-level operational categories for proceduresand service requests.
* Name: [314e Procedure Subcategories](ValueSet-procedure-category-subcategory-314e.md)
  * Description: Procedure subcategories used for more granularoperational classification.
* Name: [314e Time Exactness Units ValueSet](ValueSet-time-accuracy-units-vs.md)
  * Description: Allowed exactness/ accurate precision units for dateTime, time, and instant precision indicators.
* Name: [314e Time Precision Units ValueSet](ValueSet-time-precision-units-vs.md)
  * Description: Allowed precision units for dateTime, time, and instant precision indicators.
